# Polaris 🌟
**Do coração de Mato-Grosso para o mundo**  

Uma experiência de **Data Storytelling (Scrollytelling)** que transforma dados públicos do IBGE em uma narrativa visual cinematográfica sobre o avanço tecnológico de Mato Grosso — com excelência técnica e propósito social.

> "Construímos com intenção: dados públicos, estética de produto e engenharia para inspirar a próxima geração."
## Visão do Projeto
Transformar dados governamentais em **narrativa visual de alto nível** é um ato de transparência, educação e inovação.  
O Polaris existe para provar que dados públicos podem gerar **impacto cultural e técnico**, elevando o padrão de projetos acadêmicos para o nível esperado em times de produto de elite.

## Tecnologias
- ⚡ **Next.js 14+ (App Router)**
- 🧠 **TypeScript**
- 🎨 **Tailwind CSS + Shadcn/ui (Design System)**
- 🎬 **Framer Motion & GSAP** (animações e scrollytelling cinematográfico)
- 📊 **Recharts / Nivo** (visualização de dados)
- 🌐 **API SIDRA (IBGE)** (dados em tempo real)

## Arquitetura
O projeto segue princípios de **Clean Architecture**, separando responsabilidades para facilitar manutenção e evolução por desenvolvedores iniciantes:
- **Dados**: integrações externas e acesso à API SIDRA  
- **Domínio**: regras de negócio e orquestração de casos de uso  
- **Interface**: UI, animações e visualização interativa dos dados  

Essa divisão reduz acoplamento, melhora a legibilidade e cria um terreno sólido para aprendizado e expansão.

## Funcionalidades
- **Narrativa Interativa** com scrollytelling imersivo  
- **Consumo em Tempo Real da API IBGE**  
- **Interface Responsiva High-End** com animações cinematográficas  

## Instalação
Pré-requisitos: **Node.js 18+**

```bash
git clone https://github.com/klebr55/polaris.git
cd polaris
npm install
npm run dev
```

Acesse: `http://localhost:3000`
