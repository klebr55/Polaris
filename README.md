# Polaris 🌟
**Do coração de Mato-Grosso para o mundo**  

Uma experiência de **Data Storytelling** que transforma dados públicos do IBGE em uma narrativa visual cinematográfica sobre o avanço tecnológico de Mato Grosso — com excelência técnica e propósito social.

> "Construímos com intenção: dados públicos, estética de produto e engenharia para inspirar a próxima geração."

## Visão do Projeto
Transformar dados governamentais em **narrativa visual de alto nível** é um ato de transparência, educação e inovação.  
O Polaris existe para provar que dados públicos podem gerar **impacto cultural e técnico**, elevando o padrão de projetos acadêmicos para o nível esperado em times de produto de elite.

## Tecnologias
- ⚡ **Next.js 16+ (App Router)**
- 🧠 **TypeScript**
- 🎨 **Tailwind CSS v4** (Design System & Tokens)
- 🎬 **Framer Motion & GSAP** (animações e scrollytelling imersivo)
- 📊 **Recharts** (visualização de dados dinâmica)
- 🌐 **API SIDRA (IBGE)** (consumo de dados oficiais)

## Arquitetura
O projeto segue princípios de modularização e separação de preocupações:
- **Services**: Integrações externas e adaptadores para a API SIDRA do IBGE.
- **Modules**: Componentes complexos e seções da página (Hero, Comparativos, Cards de Dados).
- **UI**: Componentes atômicos e base do Design System.

Essa divisão reduz acoplamento, melhora a legibilidade e cria um terreno sólido para aprendizado e expansão.

## Funcionalidades
- **Narrativa Interativa** com animações de entrada e scroll imersivo.
- **Consumo em Tempo Real** via API SIDRA (IBGE) com cache e revalidação.
- **Visualização de Dados** com gráficos de alta fidelidade e performance.
- **Interface High-End** com Glassmorphism e tipografia refinada.

## Instalação
Pré-requisitos: **Node.js 18+**

```bash
git clone https://github.com/klebr55/polaris.git
cd polaris
npm install
npm run dev
```

Acesse: `http://localhost:3000`
